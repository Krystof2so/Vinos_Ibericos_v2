# Tableau des différents *commits*

```
| Type       | Quand l’utiliser                                                      | 
| ---------- | --------------------------------------------------------------------- | 
| `feat`     | Nouvelle fonctionnalité ou option ajoutée                             | 
| `fix`      | Correction d’un bug ou comportement incorrect                         |
| `refactor` | Réorganisation ou simplification du code sans changer le comportement |
| `chore`    | Tâches annexes ou maintenance, pas de modification fonctionnelle      | 
| `style`    | Ajustements d’apparence ou formatage (indentation, espaces, noms)     |
| `docs`     | Modification ou ajout dans la documentation                           |
| `test`     | Ajout ou modification de tests                                        |
| `perf`     | Optimisation du code pour performance                                 |
```
